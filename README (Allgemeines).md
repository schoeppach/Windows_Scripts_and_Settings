## ALLGEMEINES ##

### Hilfe in der Shell:
	Get-Help *resolve*
	Get-Command *resolve*
	Get-Command -Verb resolve

### Bildschirm Clear:
	cls

### Abbruch:
	strg c

### ShowWindow:
	ShowWindow
	
### IP einlenden
	ipconfig

### cache anzeigen
	ipconfig /displaydns

### cache löschen
	ipconfig /flushdns

### Windows Installation überprüfen
	sfc /scannow

### pingt bestimmte Ip im Netzwerk an
	ping 172.16.0.10

### Ethernet Statistik (Port,TCP)
	netstat -a, -b, -r

### Ip‘s im Netzwerk anzeigen 
	arp -a
	
### Ip's des Internet Dienstes
	nslookup www.heise.de
